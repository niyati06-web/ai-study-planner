# 🧠 AI Study Planner

An AI-powered web app that generates personalized study plans for any topic.

## 🚀 Features
- Enter any topic → get a day-by-day study roadmap
- Choose your level: Beginner, Intermediate, Advanced
- Set duration: 3 days to 2 months
- Powered by Groq AI (LLaMA 3.3)

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Python Flask
- AI: Groq API (LLaMA 3.3 70B)
